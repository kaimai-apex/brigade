# 07 — Go-to-Market & SEO

## The core GTM truth: this is supply-first

Unlike the booking marketplace (which needed both sides at once), Brigade is **single-sided to bootstrap.** Win chefs first. A dense, credible directory then *pulls* demand (buyers + SEO traffic) and lets you sell the employer side. Sequence:

```
Recruit chefs → directory gets dense → SEO + word-of-mouth pull buyers
 → buyers create inquiries → chefs see value → chefs pay + refer more chefs → loop
```

## Phase-by-phase acquisition

### Get the first ~100 chefs (Phases 2–4) — manual, unscalable, on purpose
- J personally recruits in the beachhead metro: DM chefs on Instagram, local culinary networks/associations, culinary-school alumni, existing private-chef Facebook/WhatsApp groups, farmers'-market and food-event presence.
- **Offer:** free Pro for life (or 12 months) for founding chefs + a hand-built profile. Make it effortless to join.
- Concierge-build their profiles for them so the directory looks full from day one (cold-start fix: never show an empty directory).

### Scale chef acquisition (Phases 5–9)
- **Referrals:** chefs invite chefs (they cluster — they sub gigs to each other). Build referral incentives in Phase 9.
- **The off-platform-lead loop:** because the profile is genuinely good, chefs send their *own* leads to it → those clients discover the directory → organic demand growth. This is the disintermediation problem inverted into a growth channel.
- **Content/SEO** (below) starts compounding.

## The SEO engine (the demand strategy)

This is Jordan's "front page of Google" instinct, done properly. SEO is the primary, lowest-CAC demand channel — but it's **slow (6–18 months)**, so it starts in Phase 2 and compounds.

### Programmatic SEO (the scalable core)
Generate indexable, high-quality pages from the chef dataset:
- `private chef in [city]`
- `[cuisine] private chef in [city]` (Italian, Japanese, vegan, BBQ…)
- `private chef for [occasion] in [city]` (dinner party, birthday, anniversary)
- `hire a private chef in [neighborhood]`

Each page = real chef listings + helpful localized content (pricing guide, what to expect, how vetting works). Thin/empty pages get penalized, so only generate a page once a city/cuisine has real supply. Architect routing + sitemaps for this from day one (see `05`).

### Editorial / content SEO
- Buyer-intent guides: "How much does a private chef cost in [city]?", "Questions to ask before hiring a private chef," "Private chef vs caterer."
- Chef-intent content (also recruits supply): "How to start a private chef business," "How to price private chef events," "Private chef contracts & deposits."
- The chef-intent content does double duty: ranks for buyers *and* pulls in chefs.

### Technical SEO
Fast SSG/ISR pages, structured data (`LocalBusiness`/`Person`/`Service`/`Review`), strong Core Web Vitals, clean canonicals — all baked into the architecture (`05`).

## Demand side beyond SEO

- **Direct B2B (Phase 8):** J sells access to estate managers, family offices, villa/yacht operators, catering agencies — the high-value buyers who today pay agencies 15–25%. Pre-sell *before* building the employer product (validate with a spreadsheet + intros first).
- **Partnerships:** luxury short-term rental operators, event planners, wine/spirits brands, high-end grocery/meal services — referral or embed deals.
- **PR / case studies:** Yhangry's "Chef X earned $Y" stories worked. Publish founding-chef success stories.

## Positioning vs incumbents in messaging

- To **chefs:** "Your professional home. Get found, look legit, run your whole business in one place — and keep 100% of what you earn." (Contrast with commission marketplaces and with looking unprofessional on Instagram.)
- To **buyers:** "Find and vet private chefs yourself, in minutes — without paying an agency thousands." (Contrast with high-touch recruiters and with the anxiety of hiring a stranger off Instagram.)

## CAC / channel priorities (cheapest → most expensive)

1. Founder-led chef recruiting + referrals (near-zero cash, high time) — **start here.**
2. SEO (low cash, slow, compounding) — **start early, harvest later.**
3. Content + community.
4. B2B direct sales to high-value buyers (J's time).
5. Paid acquisition — **only after** you know LTV and the funnel converts. Don't buy growth before the loop works.

## Leading indicators to watch

- Chef side: signups → published profiles → chefs running a real booking through tools (north star) → Pro conversion → referral rate.
- Demand side: organic impressions/clicks → inquiries per chef per month → inquiry→booking rate → paying buyers.
- If chefs sign up but never run a booking through the tools, the product isn't sticky yet — fix that before pouring on acquisition.
